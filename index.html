Open DevTools in mobile view before the first line of any new UI. Not after. Converts a rule in memory into a muscle reflex.
✍️ Constrained prompts — constraint before sending
Behavior
Every prompt should have a constraint: a specific outcome, a limit (top 3, one paragraph), or a clear decision point. Read it once before sending.
